display: flex

O Módulo de Layout Flexbox do CSS foi introduzido para facilitar o design de estruturas de layout flexíveis e responsivas, sem a necessidade de usar float ou posicionamento.

Para que o método flex do CSS funcione, envolva os elementos div com outro elemento div e atribua a ele o status de contêiner flexível (flex container).

Grid

O Módulo de Layout em Grid do CSS oferece um sistema de layout baseado em grades, com linhas e colunas, facilitando o design de páginas da web sem a necessidade de usar `float` e posicionamento.

Parece quase o mesmo que o flex, mas tem a capacidade de definir mais de uma linha e posicionar cada linha individualmente.

O método de grade do CSS exige que você envolva os elementos `div` com outro elemento `div`, atribua a ele o status de contêiner de grade (grid container) e especifique a largura de cada coluna."
