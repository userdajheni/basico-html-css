Atributos do formulário

O atributo action define a ação a ser executada quando o formulário é submetido (enviado). Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar.

O atributo target especifica onde Exibir a resposta recebida após a apresentação do formulário. Ele pode conter os seguintes valores:

  • _blank A resposta é exibida em uma nova janela ou guia
  • _self A resposta é exibida na janela atual
  • _parent A resposta é exibida no quadro pai
  • _top A resposta é exibida em todo o corpo da janela
  • framename A resposta é exibida em um iframe nomeado

O atributo method especifica o método HTTP a ser usado ao enviar os dados do formulário. Os dados de formulário podem ser enviados como variáveis de URL com method="get" ou com method="post"


O atributo autocomplete " quando está ativado, o navegador completa automaticamente os valores com base nos valores que o usuário inseriu antes.



Elementos do Formulário

  1. input
    • Atributos principais:
      • type: Define o tipo do campo de entrada, como texto, senha, número, data, etc.
      • name: Define o nome do campo para identificação no envio do formulário.
      • value: Define o valor inicial do campo.
      • Exemplos de tipos comuns: text, password, email, number, checkbox, radio.


  2. label
    • Atributo principal:
      • for: Associa o rótulo a um elemento de formulário usando o id desse elemento.
      • Benefício: Melhora a acessibilidade para leitores de tela e a usabilidade ao clicar no rótulo.


  3. select

    • Atributos principais:
      • name: Nome do campo no formulário.
      • multiple: Permite a seleção de múltiplas opções.


  4. textarea
    • Atributos principais:
      • rows e cols: Controlam a altura e largura do campo.
      • name: Nome do campo no formulário.


  5. button
    • Atributos principais:
      • type: Define o tipo de botão (submit, reset, ou apenas button).
      • onclick: Especifica um evento JavaScript a ser executado quando clicado.


  6. fieldset
    • Associado a: Geralmente usado com < legend >.


  7. legend


  8. datalist
    • Associado a: Atributo list no < input >.


  9. output
    • Atributos principais:
      • for: Associa o elemento a um ou mais elementos de formulário pelo id.
      • name: Nome para identificar o elemento no formulário.


  10. option
    • Atributos principais:
      • value: Valor enviado ao servidor quando a opção é selecionada.
      • selected: Define uma opção como selecionada por padrão.


  11. optgroup
    • Atributos principais:
      • label: Nome do grupo de opções.
